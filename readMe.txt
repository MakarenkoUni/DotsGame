Dots Game 

JavaFX-гра, де гравці створюють ланцюги з кольорових точок для отримання очок. Реалізовано з використанням Spring Boot, PostgreSQL та JavaFX.

Основні можливості

Реєстрація та Вхід

Гравці можуть реєструватися та входити в систему.

Виводяться повідомлення про помилки при неправильних даних або вже існуючих гравцях.

Ігровий процес

Гравці вибирають сусідні точки одного кольору для створення ланцюга.

Після завершення ланцюга (натисканням Enter) точки видаляються, а гравець отримує очки.

Із кожним рівнем розмір сітки збільшується.

Оновлення результатів

Після завершення гри результат оновлюється у базі даних.

Таблиця лідерів

Виводиться список гравців з найвищими результатами.

⚙️ Як працює додаток

Реєстрація/Вхід:

Новий гравець реєструється, існуючий — входить у систему.

Початок гри:

Після входу гравець переходить до ігрової сцени.

Гра:

Натискайте на сусідні точки одного кольору для створення ланцюга.

Натискайте на відповідну кнопку для завершення ходу і видалення ланцюга.

Перехід на новий рівень відбувається автоматично після закінчення кількості ходів.

Завершення гри:

Після завершення гри результат оновлюється.

Натисніть "Exit to menu" для повернення на головний екран.

Таблиця лідерів:

Переглядайте топ-10 гравців у розділі "Leaderboard".
